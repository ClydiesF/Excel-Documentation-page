V-LookUPS

V-Lookup is probably going to be one of the most used function. Vlookup allows you to grab data from another used and bring it into the current sheet that you are working on. Vlookup accomplishes this be utilizing a hook.

the V-loop-up needs to 4 things:

  1. Hook or reference value
  2. reference table
  3. index column
  4. True/False
=VLOOKUP(A5,TABLEA5:C10,2,false)

What this formual is doing is seeing if the value in cell A5 matches any cell in "Table" (our external Table sheet) if so its going to take the value thats 2 columns over and return it only if its an exact match.
